# Physical AI & Humanoid Robotics Book

This directory contains the source code for the "Physical AI & Humanoid Robotics" book, built with Docusaurus and published to GitHub Pages.

## Overview

This book provides a comprehensive guide to Physical AI - AI systems that function in reality and comprehend physical laws. Students learn to design, simulate, and deploy humanoid robots capable of natural human interactions using ROS 2, Gazebo, and NVIDIA Isaac.

## Structure

The book is organized into 4 main modules:

1. **Module 1: The Robotic Nervous System (ROS 2)** - Middleware for robot control
2. **Module 2: The Digital Twin (Gazebo & Unity)** - Physics simulation and environment building
3. **Module 3: The AI-Robot Brain (NVIDIA Isaacâ„¢)** - Advanced perception and training
4. **Module 4: Vision-Language-Action (VLA)** - Convergence of LLMs and Robotics

## Development

To run the book locally:

1. Navigate to this directory: `cd book`
2. Install dependencies: `npm install`
3. Start the development server: `npm start`

The book will be available at http://localhost:3000
